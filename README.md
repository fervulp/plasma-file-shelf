# File Shelf — Plasma 6 applet

A temporary drag-and-drop file shelf for the KDE Plasma panel, inspired by
Dropover on macOS. Drop files onto the panel icon to stage them, then drag
them out — one by one or all at once — into any other application.



https://github.com/user-attachments/assets/df044597-bc67-4db9-831d-7000140e9ef8



## Features

- **Drop to collect** — drag files onto the panel icon (or the open popup)
  to add them to the shelf; the popup opens automatically on hover or when
  a drag approaches.
- **Drag out** — drag a row into any app. If some files are selected
  (click a row or its checkbox), dragging any row carries the whole
  selection as one drop.
- **Folders become zips** — a dropped folder is automatically packed into
  a zip archive, and the archive is what gets dragged out. Handy for
  uploading folders to web forms and chats that only accept files.
- **Extension-less files get a proper extension** — dropping a file with
  no extension makes the shelf detect its type (via `file`) and stage a
  renamed copy (e.g. `photo` → `photo.png`); the original is not touched.
- **Drop images/links from the browser** — an `http(s)` URL dropped on
  the shelf is downloaded into the cache, and if the file has no
  extension, the right one (png/jpg/…) is added automatically based on
  the file's content.
- **Drop text to create a file** — dropping plain text (a selection from
  an editor or browser) creates a `.txt` file named after the first three
  words of the text.
- **Archive & encrypt via Ark** — one button opens Ark's "create archive"
  dialog (any format, optional password) for the selected files; when the
  archive is created, it replaces those files on the shelf.
- **Screen-edge drop zone** (optional) — Wayland does not let applets see
  drags happening over other windows, so as a Dropover-style trigger the
  shelf can show a subtle 6 px strip on a chosen screen edge: move a
  dragged file against it and the shelf pops up; you can also drop right
  onto the strip. Note: regular clicks on the strip are intercepted by it,
  so the option is off by default.
- **Clipboard support** — `Ctrl+A` selects everything, `Ctrl+C` copies the
  selection as `text/uri-list`, so "Paste" works in Dolphin and friends.
- **Persistent** — the shelf contents survive plasmashell restarts.
- Image thumbnails, per-file and total sizes, file-type icons.

## Settings

- **Panel icon** — pick any icon from the system theme via the standard
  KDE icon chooser (default: `document-multiple`).
- **Temporary files location** — folders dropped on the shelf are zipped
  into a `file-shelf` subfolder of the directory you choose
  (default: `~/.cache`). "Clear shelf" removes only that subfolder.

## Requirements

- KDE Plasma 6
- `zip` (packing dropped folders)
- `curl` or `wget` — only for downloading dropped browser links
- `wl-copy` (Wayland) or `xclip` (X11) — only for the Ctrl+C feature
- `ark` — only for the "pack into archive" button

## Installation

From the KDE Store: *Add widgets… → Get New Widgets → search "File Shelf"*.

From source:

```sh
kpackagetool6 --type Plasma/Applet --install .
# upgrade later with:
kpackagetool6 --type Plasma/Applet --upgrade .
```

Then add **File Shelf** to your panel.

## Security notes

The applet runs a few helper shell commands (`zip`, `stat`, `rm` inside its
own cache folder, clipboard tools) through Plasma's executable data engine.
All file paths are shell-quoted; nothing is executed from file contents,
and "Clear shelf" only ever deletes the applet's own `file-shelf` cache
subfolder.
